# Task 05 - Chat

## Goal
Implement the chat page.

## Required behavior
The chat must:
- render messages
- include prompt input
- include 3 suggested prompts
- call the backend streaming endpoint
- progressively render assistant output
- include sending/loading state
- include error state
- prevent duplicate send while streaming

## UI rules
- mobile-first
- use Ionic components
- simple and readable
- no persistence
- no markdown rendering
- no attachments
- no extra chat features

## Constraints
- do not modify backend behavior unless strictly necessary
- do not add bonus functionality
- keep implementation small

## Required response after changes
Return:
1. how the streaming flow works
2. files changed
3. exact testing steps
